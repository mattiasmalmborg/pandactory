# Pandactory

An incremental automation game where you help Dr. Redd Pawston III, a brilliant Red Panda scientist, gather resources, build automations, and construct a spaceship to escape an alien planet.

## Play Online

**[Play Pandactory](https://mattiasmalmborg.github.io/pandactory/)**

## Features

- 6 unique biomes to discover and explore
- Deep production chains with 30+ resources
- Expedition system with risk/reward mechanics
- Prestige system with permanent skill upgrades
- Offline progress support
- PWA - installable on mobile devices

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

Game runs at `http://localhost:3000`

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
